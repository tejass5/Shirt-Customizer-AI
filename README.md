# Shirt-Customizer-AI

Overview:-

1. Basic shirt customizer web app with colourpicker, and image upload functionality. 
2. Dall-e OpenAI API integrated.

**Tech Stacks Used:-**
1. Frontend:- HTML, TAILWIND-CSS, JAVASCRIPT, React.js
2. Backend:- Node, MongoDB
